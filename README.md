# Pomodoro Timer

Temporizador Pomodoro construido con React y Vite.

## Funcionalidades

- Configura los minutos de trabajo y descanso (1–60 min)
- Cuenta regresiva con cambio automatico de modo al llegar a cero
- Sonido de alerta al completar cada sesion
- Barra de progreso en tiempo real
- Guardar sesiones parciales
- Contador de sesiones completadas y tiempo total trabajado
- Historial de sesiones en pantalla

## Tecnologias

- React 18
- Vite
- CSS (sin librerias externas)

## Instalacion

```bash
npm install
npm run dev
```

## Uso

1. Ajusta los minutos de trabajo y descanso
2. Presiona **Iniciar** para comenzar
3. El timer cambia automaticamente entre trabajo y descanso
4. Usa **Guardar parcial** para registrar tiempo sin esperar al final
5. **Reiniciar** resetea todo al estado inicial
